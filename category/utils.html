How to call a function by name?

You can call a function using the standard object accessor:

root[name]()

To verify that the property is really a function you can use:

typeof root[name] === 'function'

This allows you to store a callback in a ListModel:

ListModel {
        id: itemModel
        ListElement { name: 'Music': callback: 'openMusic'}
}

This is how you would use this then:

ListView {
        model: itemModel
        delegate: ListItem {
                text: model.name
                onClicked: root[model.callback]()
        }
}
